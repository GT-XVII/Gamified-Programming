function tryGetPreferRestEnvironmentVariable ( ) = tryGetPreferRestEnvironmentVariable

Parses the value of the environment variable FIRESTORE_PREFER_REST, and returns a value indicating if the environment variable enables or disables preferRest.

This function will warn to the console if the environment variable is set to an unsupported value.

Returns
