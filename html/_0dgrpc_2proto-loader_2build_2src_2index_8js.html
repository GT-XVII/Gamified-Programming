◆ load()

function load ( filename,
options )

Load a .proto file with the specified options.

Parameters
filenameOne or multiple file paths to load. Can be an absolute path or relative to an include path.
options.keepCasePreserve field names. The default is to change them to camel case.
options.longsThe type that should be used to represent long values. Valid options are Number and String. Defaults to a Long object type from a library.
options.enumsThe type that should be used to represent enum values. The only valid option is String. Defaults to the numeric value.
options.bytesThe type that should be used to represent bytes values. Valid options are Array and String. The default is to use Buffer.
options.defaultsSet default values on output objects. Defaults to false.
options.arraysSet empty arrays for missing array values even if defaults is false. Defaults to false.
options.objectsSet empty objects for missing object values even if defaults is false. Defaults to false.
options.oneofsSet virtual oneof properties to the present field's name
options.jsonRepresent Infinity and NaN as strings in float fields, and automatically decode google.protobuf.Any values.
options.includeDirsPaths to search for imported .proto files.
