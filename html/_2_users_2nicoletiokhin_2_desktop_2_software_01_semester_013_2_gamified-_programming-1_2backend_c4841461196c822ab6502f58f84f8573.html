Creates and returns a new [Filter]Filter, which can be applied to [Query.where()]Query#where, [Filter.or()]Filter#or, or [Filter.and()]Filter#and. When applied to a [Query]Query it requires that documents must contain the specified field and that its value should satisfy the relation constraint provided.

Parameters
{string|FieldPath}fieldPath The name of a property value to compare.
{string}opStr A comparison operation in the form of a string. Acceptable operator strings are "<", "<=", "==", "!=", ">=", ">", "array-contains", "in", "not-in", and "array-contains-any".
{*}value The value to which to compare the field for inclusion in a query.
Returns
{Filter} The created Filter.
let collectionRef = firestore.collection('col');
collectionRef.where(Filter.where('foo', '==', 'bar')).get().then(querySnapshot => {
querySnapshot.forEach(documentSnapshot => {
console.log(`Found document at ${documentSnapshot.ref.path}`);
});
});
