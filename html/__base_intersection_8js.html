function baseIntersection ( arrays,

private

The base implementation of methods like _.intersection, without support for iteratee shorthands, that accepts an array of arrays to inspect.

Parameters
{Array}arrays The arrays to inspect.
{Function}[iteratee] The iteratee invoked per element.
{Function}[comparator] The comparator invoked per element.
Returns
{Array} Returns the new array of shared values.
