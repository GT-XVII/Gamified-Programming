A specialized version of baseIsEqualDeep for arrays with support for partial deep comparisons.

Parameters
{Array}array The array to compare.
{Array}other The other array to compare.
{number}bitmask The bitmask flags. See baseIsEqual for more details.
{Function}customizer The function to customize comparisons.
{Function}equalFunc The function to determine equivalents of values.
{Object}stack Tracks traversed array and other objects.
Returns
{boolean} Returns true if the arrays are equivalent, else false.
