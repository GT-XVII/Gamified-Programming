function initCloneByTag ( object,
tag,
isDeep )
private

Initializes an object clone based on its toStringTag.

Note: This function only supports cloning values with tags of Boolean, Date, Error, Map, Number, RegExp, Set, or String.

Parameters
{Object}object The object to clone.
{string}tag The toStringTag of the object to clone.
{boolean}[isDeep] Specify a deep clone.
Returns
{Object} Returns the initialized clone.
