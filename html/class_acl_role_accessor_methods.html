Detailed Description

Attach functionality to a Storage.acl instance. This will add an object for each role group (owners, readers, and writers), with each object containing methods to add or delete a type of entity.

As an example, here are a few methods that are created.

myBucket.acl.readers.deleteGroup('groupId', function(err) {});

myBucket.acl.owners.addUser('email.nosp@m.@exa.nosp@m.mple..nosp@m.com', function(err, acl) {});

myBucket.acl.writers.addDomain('example.com', function(err, acl) {});
