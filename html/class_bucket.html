◆ makeAllFilesPublicPrivate_() [1/2]

Bucket::makeAllFilesPublicPrivate_ ( optionsOrCallback,
callback )
inlineprivate

Iterate over all of a bucket's files, calling file.makePublic() (public) or file.makePrivate() (private) on each.

Operations are performed in parallel, up to 10 at once. The first error breaks the loop, and will execute the provided callback with it. Specify { force: true } to suppress the errors.

Parameters
{MakeAllFilesPublicPrivateOptions}[options] Configuration options.
{boolean}[options.force] Suppress errors until all files have been processed.
{boolean}[options.private] Make files private.
{boolean}[options.public] Make files public.
{string}[options.userProject] The ID of the project which will be billed for the request.
{MakeAllFilesPublicPrivateCallback}callback Callback function.
Returns
{Promise<MakeAllFilesPublicPrivateResponse>}
