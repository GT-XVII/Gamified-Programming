◆ listen()

CompletionListener::listen ( commands,
abortSignal )
inline

Given a list of commands, wait for all of them to exit and then evaluate their exit codes.

Returns
A Promise that resolves if the success condition is met, or rejects otherwise. In either case, the value is a list of close events for commands that spawned. Commands that didn't spawn are filtered out.
