Detailed Description

A CompositeFilter is used to narrow the set of documents returned by a Firestore query by performing the logical OR or AND of multiple Filterss. CompositeFilterss are created by invoking Filter#or or Filter#and and can then be passed to Query#where to create a new query instance that also contains the CompositeFilter.
