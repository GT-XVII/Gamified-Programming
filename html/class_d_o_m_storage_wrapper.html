See the License for the specific language governing permissions and limitations under the License. Wraps a DOM Storage object and:

  • automatically encode objects as JSON strings before storing them to allow us to store arbitrary types.
  • prefixes names with "firebase:" to avoid collisions with app data.

We automatically (see storage.js) create two such wrappers, one for sessionStorage, and one for localStorage.
