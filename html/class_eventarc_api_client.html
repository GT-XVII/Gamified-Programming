◆ publish()

async EventarcApiClient::publish ( events)
inline

Publishes provided events to this channel. If channel was created with allowedEventsTypes and event type is not on that list, the event is ignored.

The following CloudEvent fields are auto-populated if not set:

  • specversion - 1.0
  • id - uuidv4()
  • source - populated with process.env.EVENTARC_CLOUD_EVENT_SOURCE and if not set an error is thrown.
Parameters
events- CloudEvent to publish to the channel.
