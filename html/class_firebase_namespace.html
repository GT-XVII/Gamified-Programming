◆ initializeApp()

FirebaseNamespace::initializeApp ( options,
appName )
inline

Initializes the FirebaseApp instance.

Parameters
options- Optional options for the FirebaseApp instance. If none present will try to initialize from the FIREBASE_CONFIG environment variable. If the environment variable contains a string that starts with '{' it will be parsed as JSON, otherwise it will be assumed to be pointing to a file.
appName- Optional name of the FirebaseApp instance.
Returns
A new FirebaseApp instance.
