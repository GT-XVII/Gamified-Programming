Parameters
{Object}request The request object that will be sent.
{string}request.name The resource that owns the locations collection, if applicable.

{number}request.pageSize The standard list page size.

{object}[options] Call options. See https://googleapis.dev/nodejs/google-gax/latest/interfaces/CallOptions.html|CallOptions for more details.
Returns
{Promise} - The promise which resolves to an array. The first element of the array is Array of [Location]google.cloud.location.Location. The client library will perform auto-pagination by default: it will call the API as many times as needed and will merge results from all the pages into this array. Note that it can affect your quota. We recommend using listLocationsAsync() method described below for async iteration which you can stop as needed. Please see the documentation for more details and examples.
