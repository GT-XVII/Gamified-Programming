Detailed Description

Developer Documentation

paginator is used to auto-paginate nextQuery methods as well as streamifying them.

Before:

search.query('done=true', function(err, results, nextQuery) { search.query(nextQuery, function(err, results, nextQuery) {}); });

After:

search.query('done=true', function(err, results) {});

Methods to extend should be written to accept callbacks and return a nextQuery.
