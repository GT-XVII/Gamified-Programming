◆ getOrInitializeService()

Provider::getOrInitializeService ( { instanceIdentifier, options={} })
inline

Invoke onInit listeners. Note this.component.onInstanceCreated is different, which is used by the component creator, while onInit listeners are registered by consumers of the provider.

Order is important onInstanceCreated() should be called after this.instances.set(instanceIdentifier, instance); which makes isInitialized() return true.
