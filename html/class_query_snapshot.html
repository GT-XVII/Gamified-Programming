Detailed Description

A QuerySnapshot contains zero or more [QueryDocumentSnapshot]QueryDocumentSnapshot objects representing the results of a query. The documents can be accessed as an array via the [documents]QuerySnapshot#documents property or enumerated using the [forEach]QuerySnapshot#forEach method. The number of documents can be determined via the [empty]QuerySnapshot#empty and [size]QuerySnapshot#size properties.
