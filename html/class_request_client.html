◆ getRetryDelayMillis()

RequestClient::getRetryDelayMillis ( retryAttempts,
err )
inline

Checks if a failed request is eligible for a retry, and if so returns the duration to wait before initiating the retry.

Parameters
retryAttempts- Number of retries completed up to now.
err- The last encountered error.
Returns
A 2-tuple where the 1st element is the duration to wait before another retry, and the 2nd element is a boolean indicating whether the request is eligible for a retry or not.
