Detailed Description

ResourceCollector class implements asynchronous logic of calling the API call that supports pagination, page by page, collecting all resources (up to maxResults) in the array.

Usage: const resourceCollector = new ResourceCollector(apiCall, maxResults); // -1 for unlimited resourceCollector.processAllPages(request).then(resources => ...);
