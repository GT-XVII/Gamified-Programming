A class representing a connection to a single backend.

Parameters
channelTargetThe target string for the channel as a whole
subchannelAddressThe address for the backend that this subchannel will connect to
optionsThe channel options, plus any specific subchannel options for this subchannel
credentialsThe channel credentials used to establish this connection

The subchannel's current connectivity state. Invariant: session === null if and only if connectivityState is IDLE or TRANSIENT_FAILURE.

The underlying http2 session used to make requests.

Indicates that the subchannel should transition from TRANSIENT_FAILURE to CONNECTING instead of IDLE when the backoff timeout ends.

A list of listener functions that will be called whenever the connectivity state changes. Will be modified by addConnectivityStateListener and removeConnectivityStateListener

Tracks channels and subchannel pools with references to this subchannel
