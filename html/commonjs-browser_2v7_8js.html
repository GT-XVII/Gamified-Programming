UUID V7 - Unix Epoch time-based UUID

The IETF has published RFC9562, introducing 3 new UUID versions (6,7,8). This implementation of V7 is based on the accepted, though not yet approved, revisions.

RFC 9562:https://www.rfc-editor.org/rfc/rfc9562.html Universally Unique IDentifiers (UUIDs)

Sample V7 value: https://www.rfc-editor.org/rfc/rfc9562.html#name-example-of-a-uuidv7-value

Monotonic Bit Layout: RFC rfc9562.6.2 Method 1, Dedicated Counter Bits ref: https://www.rfc-editor.org/rfc/rfc9562.html#section-6.2-5.1

seq is a 31 bit serialized counter; comprised of 12 bit seq_hi and 19 bit seq_low, and randomly initialized upon timestamp change. 31 bit counter size was selected as any bitwise operations in node are done as signed 32 bit ints. we exclude the sign bit.
