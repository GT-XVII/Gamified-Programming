◆ getRelativeTimeout()

function getRelativeTimeout ( deadline)

Get the timeout value that should be passed to setTimeout now for the timer to end at the deadline. For any deadline before now, the timer should end immediately, represented by a value of 0. For any deadline more than MAX_TIMEOUT_TIME milliseconds in the future, a timer cannot be set that will end at that time, so it is treated as infinitely far in the future.

Parameters
deadline
Returns
