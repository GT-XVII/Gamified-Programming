An options object to configure throttling on BulkWriter.

Whether to disable or configure throttling. By default, throttling is enabled. throttling can be set to either a boolean or a config object. Setting it to true will use default values. You can override the defaults by setting it to false to disable throttling, or by setting the config values to enable throttling with the provided values.

throttling. Throttling is enabled by default, if the field is set to true or if any custom throttling options are provided. { initialOpsPerSecond: number } sets the initial maximum number of operations per second allowed by the throttler. If initialOpsPerSecond is not set, the default is 500 operations per second. { maxOpsPerSecond: number } sets the maximum number of operations per second allowed by the throttler. If maxOpsPerSecond is not set, no maximum is enforced.
