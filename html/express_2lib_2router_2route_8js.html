Add a handler for all HTTP verbs to this route.

Behaves just like middleware and can respond or call next to continue processing.

You can use multiple .all call to add multiple handlers.

function check_something(req, res, next){ next(); };

function validate_user(req, res, next){ next(); };

route .all(validate_user) .all(check_something) .get(function(req, res, next){ res.send('hello world'); });
