function findProjectId ( app)

Determines the Google Cloud project ID associated with a Firebase app. This method first checks if a project ID is explicitly specified in either the Firebase app options, credentials or the local environment in that order. If no explicit project ID is configured, but the SDK has been initialized with ComputeEngineCredentials, this method attempts to discover the project ID from the local metadata service.

Parameters
app- A Firebase app to get the project ID from.
Returns
A project ID string or null.
