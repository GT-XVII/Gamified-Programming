Derives a key from a password.

Parameters
pthe password as a binary-encoded string of bytes.
sthe salt as a binary-encoded string of bytes.
cthe iteration count, a positive integer.
dkLenthe intended length, in bytes, of the derived key, (max: 2^32 - 1) * hash length of the PRF.
[md]the message digest (or algorithm identifier as a string) to use in the PRF, defaults to SHA-1.
[callback(err,key)]presence triggers asynchronous version, called once the operation completes.
Returns
the derived key, as a binary-encoded string of bytes, for the synchronous version (if no callback is specified).
