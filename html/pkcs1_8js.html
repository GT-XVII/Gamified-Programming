◆ decode_rsa_oaep

pkcs1 decode_rsa_oaep

Decode the given RSAES-OAEP encoded message (EM) using key, with optional label (L).

This method does not perform RSA decryption, it only decodes the message using RSAES-OAEP.

Parameters
keythe RSA key to use.
emthe encoded message to decode.
optionsthe options to use: label an optional label to use. md the message digest object to use for OAEP, undefined for SHA-1. mgf1 optional mgf1 parameters: md the message digest object to use for MGF1.
Returns
the decoded message bytes.

◆ encode_rsa_oaep

pkcs1 encode_rsa_oaep

Encode the given RSAES-OAEP message (M) using key, with optional label (L) and seed.

This method does not perform RSA encryption, it only encodes the message using RSAES-OAEP.

Parameters
keythe RSA key to use.
messagethe message to encode.
optionsthe options to use: label an optional label to use. seed the seed to use. md the message digest object to use, undefined for SHA-1. mgf1 optional mgf1 parameters: md the message digest object to use for MGF1.
Returns
the encoded message bytes.
