◆ generateProbablePrime

prime generateProbablePrime

Generates a random probable prime with the given number of bits.

Alternative algorithms can be specified by name as a string or as an object with custom options like so:

{ name: 'PRIMEINC', options: { maxBlockTime: <the maximum amount of time to block the main thread before allowing I/O other JS to run>, millerRabinTests: <the number of miller-rabin tests to run>, workerScript: <the worker script URL>, workers: <the number of web workers (if supported) to use, -1 to use estimated cores minus one>. workLoad: the size of the work load, ie: number of possible prime numbers for each web worker to check per work assignment, (default: 100). } }

Parameters
bitsthe number of bits for the prime number.
optionsthe options to use. [algorithm] the algorithm to use (default: 'PRIMEINC'). [prng] a custom crypto-secure pseudo-random number generator to use, that must define "getBytesSync".
Returns
callback(err, num) called once the operation completes.
