Definition index.js:13

A generic rate limiter. Underneath the hood, this uses a token bucket plus an additional check to limit how many tokens we can remove each interval.

Author
John Hurliman jhurl.nosp@m.iman.nosp@m.@jhur.nosp@m.lima.nosp@m.n.org
Parameters
{Number}tokensPerInterval Maximum number of tokens that can be removed at any given moment and over the course of one interval.
{String|Number}interval The interval length in milliseconds, or as one of the following strings: 'second', 'minute', 'hour', day'.
{Boolean}fireImmediately Optional. Whether or not the callback will fire immediately when rate limiting is in effect (default is false).
