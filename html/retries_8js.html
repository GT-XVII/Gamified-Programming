Creates a function equivalent to func, but that retries on certain exceptions.

Parameters

{RetryOptions}retry - Configures the exceptions upon which the function eshould retry, and the parameters to the exponential backoff retry algorithm.
{GRPCCallOtherArgs}otherArgs - the additional arguments to be passed to func.

{SimpleCallbackFunction} A function that will retry.

Equivalent to func, but retries upon transient failure.

Retrying is done through an exponential backoff algorithm configured by the options in retry.
