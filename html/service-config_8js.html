◆ extractAndSelectServiceConfig()

function extractAndSelectServiceConfig ( txtRecord,
percentage )

Find the "grpc_config" record among the TXT records, parse its value as JSON, validate its contents, and select a service config with selection fields that all match this client. Most of these steps can fail with an error; the caller must handle any errors thrown this way.

Parameters
txtRecordThe TXT record array that is output from a successful call to dns.resolveTxt
percentageA number chosen from the range [0, 100) that is used to select which config to use
Returns
The service configuration to use, given the percentage value, or null if the service config data has a valid format but none of the options match the current client.
