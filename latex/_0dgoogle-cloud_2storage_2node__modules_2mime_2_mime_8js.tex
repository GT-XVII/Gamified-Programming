Define mimetype -\/\texorpdfstring{$>$}{>} extension mappings. Each key is a mime-\/type that maps to an array of extensions associated with the type. The first extension is used as the default extension for the type.

e.\+g. mime.\+define(\{\textquotesingle{}audio/ogg\textquotesingle{}, \mbox{[}\textquotesingle{}oga\textquotesingle{}, \textquotesingle{}ogg\textquotesingle{}, \textquotesingle{}spx\textquotesingle{}\mbox{]}\});

If a type declares an extension that has already been defined, an error will be thrown. To suppress this error and force the extension to be associated with the new type, pass {\ttfamily force}=true. Alternatively, you may prefix the extension with "{}\texorpdfstring{$\ast$}{*}"{} to map the type to extension, without mapping the extension to the type.

e.\+g. mime.\+define(\{\textquotesingle{}audio/wav\textquotesingle{}, \mbox{[}\textquotesingle{}wav\textquotesingle{}\mbox{]}\}, \{\textquotesingle{}audio/x-\/wav\textquotesingle{}, \mbox{[}\textquotesingle{}\texorpdfstring{$\ast$}{*}wav\textquotesingle{}\mbox{]}\});
