function \+\_\+make\+Compatibility\+Check (\begin{DoxyParamCaption}\item[{}]{own\+Version}{}\end{DoxyParamCaption})}

Create a function to test an API version to see if it is compatible with the provided own\+Version.

The returned function has the following semantics\+:
\begin{DoxyItemize}
\item Exact match is always compatible
\item Major versions must match exactly
\begin{DoxyItemize}
\item 1.\+x package cannot use global 2.\+x package
\item 2.\+x package cannot use global 1.\+x package
\end{DoxyItemize}
\item The minor version of the API module requesting access to the global API must be less than or equal to the minor version of this API
\begin{DoxyItemize}
\item 1.\+3 package may use 1.\+4 global because the later global contains all functions 1.\+3 expects
\item 1.\+4 package may NOT use 1.\+3 global because it may try to call functions which don\textquotesingle{}t exist on 1.\+3
\end{DoxyItemize}
\item If the major version is 0, the minor version is treated as the major and the patch is treated as the minor
\item Patch and build tag differences are not considered at this time
\end{DoxyItemize}
