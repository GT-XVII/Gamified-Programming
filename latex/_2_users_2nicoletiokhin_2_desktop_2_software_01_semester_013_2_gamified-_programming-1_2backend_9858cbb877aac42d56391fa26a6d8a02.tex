If {\ttfamily resolver} is provided, it determines the cache key for storing the result based on the arguments provided to the memoized function. By default, the first argument provided to the memoized function is used as the map cache key. The {\ttfamily func} is invoked with the {\ttfamily this} binding of the memoized function.

{\bfseries{Note\+:}} The cache is exposed as the {\ttfamily cache} property on the memoized function. Its creation may be customized by replacing the {\ttfamily \+\_\+.\+memoize.\+Cache} constructor with one whose instances implement the \href{http://ecma-international.org/ecma-262/7.0/\#sec-properties-of-the-map-prototype-object}{\texttt{ {\ttfamily Map}}} method interface of {\ttfamily clear}, {\ttfamily delete}, {\ttfamily get}, {\ttfamily has}, and {\ttfamily set}.
