function merge\+Data (\begin{DoxyParamCaption}\item[{}]{data}{, }\item[{}]{source}{}\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [private]}}

Merges the function metadata of {\ttfamily source} into {\ttfamily data}.

Merging metadata reduces the number of wrappers used to invoke a function. This is possible because methods like {\ttfamily \+\_\+.\+bind}, {\ttfamily \+\_\+.\+curry}, and {\ttfamily \+\_\+.\+partial} may be applied regardless of execution order. Methods like {\ttfamily \+\_\+.\+ary} and {\ttfamily \+\_\+.\+rearg} modify function arguments, making the order in which they are executed important, preventing the merging of metadata. However, we make an exception for a safe combined case where curried functions have {\ttfamily \+\_\+.\+ary} and or {\ttfamily \+\_\+.\+rearg} applied.
