See NIST SP-\/800-\/38D 6.\+3 (Algorithm 1). This function performs Galois field multiplication. The field, GF(2\texorpdfstring{$^\wedge$}{\string^}128), is defined by the polynomial\+:

x\texorpdfstring{$^\wedge$}{\string^}128 + x\texorpdfstring{$^\wedge$}{\string^}7 + x\texorpdfstring{$^\wedge$}{\string^}2 + x + 1

Which is represented in little-\/endian binary form as\+: 11100001 (0xe1). When the value of a coefficient is 1, a bit is set. The value R, is the concatenation of this value and 120 zero bits, yielding a 128-\/bit value which matches the block size.

This function will multiply two elements (vectors of bytes), X and Y, in the field GF(2\texorpdfstring{$^\wedge$}{\string^}128). The result is initialized to zero. For each bit of X (out of 128), x\+\_\+i, if x\+\_\+i is set, then the result is multiplied (XOR\textquotesingle{}d) by the current value of Y. For each bit, the value of Y will be raised by a power of x (multiplied by the polynomial x). This can be achieved by shifting Y once to the right. If the current value of Y, prior to being multiplied by x, has 0 as its LSB, then it is a 127th degree polynomial. Otherwise, we must divide by R after shifting to find the remainder.
