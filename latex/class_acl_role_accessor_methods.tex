\doxysubsection{Detailed Description}
Attach functionality to a {\ttfamily \doxylink{}{Storage.\+acl}} instance. This will add an object for each role group (owners, readers, and writers), with each object containing methods to add or delete a type of entity.

As an example, here are a few methods that are created.

my\+Bucket.\+acl.\+readers.\+delete\+Group(\textquotesingle{}group\+Id\textquotesingle{}, function(err) \{\});

my\+Bucket.\+acl.\+owners.\+add\+User(\textquotesingle{}\href{mailto:email@example.com}{\texttt{ email@example.\+com}}\textquotesingle{}, function(err, acl) \{\});

my\+Bucket.\+acl.\+writers.\+add\+Domain(\textquotesingle{}example.\+com\textquotesingle{}, function(err, acl) \{\});
