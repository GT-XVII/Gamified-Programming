\doxysubsection{Detailed Description}
An HTTP client that can be used to make batch requests. This client is not tied to any service (FCM or otherwise). Therefore it can be used to make batch requests to any service that allows it. If this requirement ever arises we can move this implementation to the utils module where it can be easily shared among other modules.
