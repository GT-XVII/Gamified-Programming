\doxysubsection{Detailed Description}
\doxylink{class_children_node}{Children\+Node} is a class for storing internal nodes in a \doxylink{class_data_snapshot}{Data\+Snapshot} (i.\+e. nodes with children). It implements \doxylink{class_node}{Node} and stores the list of children in the children property, sorted by child name.
