Completion\+Listener\+::listen (\begin{DoxyParamCaption}\item[{}]{commands}{, }\item[{}]{abort\+Signal}{}\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [inline]}}

Given a list of commands, wait for all of them to exit and then evaluate their exit codes.

\begin{DoxyReturn}{Returns}
\doxylink{class_a}{A} Promise that resolves if the success condition is met, or rejects otherwise. In either case, the value is a list of close events for commands that spawned. Commands that didn\textquotesingle{}t spawn are filtered out. 
\end{DoxyReturn}
