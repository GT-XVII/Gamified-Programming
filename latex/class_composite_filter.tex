\doxysubsection{Detailed Description}
\doxylink{class_a}{A} {\ttfamily \doxylink{class_composite_filter}{Composite\+Filter}} is used to narrow the set of documents returned by a \doxylink{class_firestore}{Firestore} query by performing the logical OR or AND of multiple {\ttfamily \doxylink{}{Filters}}s. {\ttfamily Composite\+Filters}s are created by invoking {\ttfamily \doxylink{class_filter_a206bac0e574a0b6a4c09747a50d0bc52}{Filter\#or}} or {\ttfamily \doxylink{class_filter_adf8b4ed4a08705db57bb2e4c483336e0}{Filter\#and}} and can then be passed to {\ttfamily \doxylink{class_query_a09b549eea59e9d2d0d81b07b1b34bb77}{Query\#where}} to create a new query instance that also contains the {\ttfamily \doxylink{class_composite_filter}{Composite\+Filter}}.
