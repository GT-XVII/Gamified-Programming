See the License for the specific language governing permissions and limitations under the License. Wraps a DOM \doxylink{class_storage}{Storage} object and\+:
\begin{DoxyItemize}
\item automatically encode objects as JSON strings before storing them to allow us to store arbitrary types.
\item prefixes names with "{}firebase\+:"{} to avoid collisions with app data.
\end{DoxyItemize}

We automatically (see storage.\+js) create two such wrappers, one for session\+Storage, and one for local\+Storage.
