\doxysubsection{Detailed Description}
\doxylink{class_a}{A} \doxylink{class_document_reference}{Document\+Reference} refers to a document location in a \doxylink{class_firestore}{Firestore} database and can be used to write, read, or listen to the location. The document at the referenced location may or may not exist. \doxylink{class_a}{A} \doxylink{class_document_reference}{Document\+Reference} can also be used to create a \mbox{[}\doxylink{class_collection_reference}{Collection\+Reference}\mbox{]}{\ttfamily \doxylink{class_collection_reference}{Collection\+Reference}} to a subcollection.
