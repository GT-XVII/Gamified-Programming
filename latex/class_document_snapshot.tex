\doxysubsection{Detailed Description}
\doxylink{class_a}{A} \doxylink{class_document_snapshot}{Document\+Snapshot} is an immutable representation for a document in a \doxylink{class_firestore}{Firestore} database. The data can be extracted with \mbox{[}\doxylink{class_document_snapshot_a2779504a2af8f1a97e5431c9c40d4e5d}{data()}\mbox{]}{\ttfamily \doxylink{class_document_snapshot_a2779504a2af8f1a97e5431c9c40d4e5d}{Document\+Snapshot\#data}} or \mbox{[}\doxylink{response_8js_a10d7f5ef6343bd4575249458cf93dd3e}{get(field\+Path)}\mbox{]}{\ttfamily \doxylink{class_document_snapshot_a57b0cb444662b5793585625a5b357cd2}{Document\+Snapshot\#get}} to get a specific field.

For a \doxylink{class_document_snapshot}{Document\+Snapshot} that points to a non-\/existing document, any data access will return \textquotesingle{}undefined\textquotesingle{}. You can use the \mbox{[}exists\mbox{]}{\ttfamily \doxylink{class_document_snapshot_aebd8a2962c04eddc15d1dd114f9b5cb0}{Document\+Snapshot\#exists}} property to explicitly verify a document\textquotesingle{}s existence.
