\doxysubsection{Detailed Description}
\doxylink{class_a}{A} {\ttfamily \doxylink{class_filter}{Filter}} represents a restriction on one or more field values and can be used to refine the results of a {\ttfamily \doxylink{class_query}{Query}}. {\ttfamily Filters}s are created by invoking {\ttfamily \doxylink{class_filter_af6be2934075d0920033f1d57a1608b11}{Filter\#where}}, {\ttfamily \doxylink{class_filter_a206bac0e574a0b6a4c09747a50d0bc52}{Filter\#or}}, or {\ttfamily \doxylink{class_filter_adf8b4ed4a08705db57bb2e4c483336e0}{Filter\#and}} and can then be passed to {\ttfamily \doxylink{class_query_a09b549eea59e9d2d0d81b07b1b34bb77}{Query\#where}} to create a new {\ttfamily \doxylink{class_query}{Query}} instance that also contains this {\ttfamily \doxylink{class_filter}{Filter}}. \begin{Desc}
