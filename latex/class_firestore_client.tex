\doxysubsection{Detailed Description}
The Cloud \doxylink{class_firestore}{Firestore} service.

Cloud \doxylink{class_firestore}{Firestore} is a fast, fully managed, serverless, cloud-\/native No\+SQL document database that simplifies storing, syncing, and querying data for your mobile, web, and IoT apps at global scale. Its client libraries provide live synchronization and offline support, while its security features and integrations with Firebase and Google Cloud Platform accelerate building truly serverless apps.
