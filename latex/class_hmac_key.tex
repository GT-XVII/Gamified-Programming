\doxysubsection{Detailed Description}
An \doxylink{class_hmac_key}{Hmac\+Key} object contains metadata of an HMAC key created from a service account through the {\ttfamily \doxylink{class_storage}{Storage}} client using {\ttfamily \doxylink{class_storage_ade8f0d2a48d1295054dc5893e8688b48}{Storage\#create\+Hmac\+Key}}.
