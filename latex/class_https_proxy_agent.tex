\doxysubsection{Detailed Description}
The {\ttfamily \doxylink{class_https_proxy_agent}{Https\+Proxy\+Agent}} implements an HTTP \doxylink{class_agent}{Agent} subclass that connects to the specified "{}\+HTTP(s) proxy server"{} in order to proxy HTTPS requests.

Outgoing HTTP requests are first tunneled through the proxy server using the {\ttfamily CONNECT} HTTP request method to establish a connection to the proxy server, and then the proxy server connects to the destination target and issues the HTTP request from the proxy server.

{\ttfamily https\+:} requests have their socket connection upgraded to TLS once the connection to the proxy server has been established.
