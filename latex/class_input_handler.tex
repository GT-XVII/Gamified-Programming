\doxysubsection{Detailed Description}
Sends input from concurrently through to commands.

Input can start with a command identifier, in which case it will be sent to that specific command. For instance, {\ttfamily 0\+:bla} will send {\ttfamily bla} to command at index {\ttfamily 0}, and {\ttfamily server\+:stop} will send {\ttfamily stop} to command with name {\ttfamily server}.

If the input doesn\textquotesingle{}t start with a command identifier, it is then always sent to the default target.
