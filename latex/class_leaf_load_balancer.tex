\doxysubsection{Detailed Description}
This class handles the leaf load balancing operations for a single endpoint. It is a thin wrapper around a \doxylink{class_pick_first_load_balancer}{Pick\+First\+Load\+Balancer} with a different API that more closely reflects how it will be used as a leaf balancer.
