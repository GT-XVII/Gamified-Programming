\doxysubsection{Detailed Description}
\doxylink{class_leaf_node}{Leaf\+Node} is a class for storing leaf nodes in a \doxylink{class_data_snapshot}{Data\+Snapshot}. It implements \doxylink{class_node}{Node} and stores the value of the node (a string, number, or boolean) accessible via \doxylink{class_leaf_node_a84eae83d25ded7e218e0586e0a99fba5}{get\+Value()}.
