\doxysubsection{Detailed Description}
An immutable object representing a parsed path. It\textquotesingle{}s immutable so that you can pass them around to other functions without worrying about them changing it.

An abstract class representing a \doxylink{class_firestore}{Firestore} path.

Subclasses have to implement {\ttfamily \doxylink{split_8js_afeb1a7ef0d383073a030166165cd4b99}{split()}} and {\ttfamily canonical\+String()}.
