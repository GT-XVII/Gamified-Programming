\doxysubsection{Detailed Description}
\doxylink{class_a}{A} \doxylink{class_query_document_snapshot}{Query\+Document\+Snapshot} contains data read from a document in your \doxylink{class_firestore}{Firestore} database as part of a query. The document is guaranteed to exist and its data can be extracted with \mbox{[}\doxylink{class_query_document_snapshot_aa38e6a745886dcfed573fe6e12925b66}{data()}\mbox{]}{\ttfamily \doxylink{class_query_document_snapshot_aa38e6a745886dcfed573fe6e12925b66}{Query\+Document\+Snapshot\#data}} or \mbox{[}\doxylink{response_8js_a10d7f5ef6343bd4575249458cf93dd3e}{get()}\mbox{]}{\ttfamily \doxylink{class_document_snapshot_a57b0cb444662b5793585625a5b357cd2}{Document\+Snapshot\#get}} to get a specific field.

\doxylink{class_a}{A} \doxylink{class_query_document_snapshot}{Query\+Document\+Snapshot} offers the same API surface as a {\ttfamily \doxylink{class_document_snapshot}{Document\+Snapshot}}. Since query results contain only existing documents, the \mbox{[}exists\mbox{]}{\ttfamily \doxylink{class_document_snapshot_aebd8a2962c04eddc15d1dd114f9b5cb0}{Document\+Snapshot\#exists}} property will always be true and \mbox{[}\doxylink{class_query_document_snapshot_aa38e6a745886dcfed573fe6e12925b66}{data()}\mbox{]}{\ttfamily \doxylink{class_query_document_snapshot_aa38e6a745886dcfed573fe6e12925b66}{Query\+Document\+Snapshot\#data}} will never return \textquotesingle{}undefined\textquotesingle{}.
