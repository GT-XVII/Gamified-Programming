\doxysubsection{Detailed Description}
Internal class representing custom \doxylink{class_query}{Query} options.

These options are immutable. Modified options can be created using {\ttfamily \doxylink{class_query_options_a2178aef7f7b301fe7cc2013163645e1c}{with()}}.
