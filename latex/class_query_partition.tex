\doxysubsection{Detailed Description}
\doxylink{class_a}{A} split point that can be used in a query as a starting and/or end point for the query results. The cursors returned by {\ttfamily \doxylink{class_query_partition_afca777765bc65ebc67f7955e43c73330}{start\+At}} and {\ttfamily \doxylink{class_query_partition_ac2486845c0f9e579803eed203c2a5b55}{end\+Before}} can only be used in a query that matches the constraint of query that produced this partition.
