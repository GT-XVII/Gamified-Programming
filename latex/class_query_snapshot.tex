\doxysubsection{Detailed Description}
\doxylink{class_a}{A} \doxylink{class_query_snapshot}{Query\+Snapshot} contains zero or more \mbox{[}\doxylink{class_query_document_snapshot}{Query\+Document\+Snapshot}\mbox{]}{\ttfamily \doxylink{class_query_document_snapshot}{Query\+Document\+Snapshot}} objects representing the results of a query. The documents can be accessed as an array via the \mbox{[}documents\mbox{]}{\ttfamily \doxylink{}{Query\+Snapshot\#documents}} property or enumerated using the \mbox{[}for\+Each\mbox{]}{\ttfamily \doxylink{class_query_snapshot_a7cb3bbed4e6b398fb95468194884c0e0}{Query\+Snapshot\#for\+Each}} method. The number of documents can be determined via the \mbox{[}empty\mbox{]}{\ttfamily \doxylink{class_query_snapshot_a3a2200080de64e727524012aa9441a5f}{Query\+Snapshot\#empty}} and \mbox{[}size\mbox{]}{\ttfamily \doxylink{class_query_snapshot_a2f087d8d024b81215234eeca6a766d18}{Query\+Snapshot\#size}} properties.
