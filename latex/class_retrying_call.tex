The offset of message indices in the write\+Buffer. For example, if write\+Buffer\+Offset is 10, message 10 is in write\+Buffer\mbox{[}0\mbox{]} and message 15 is in write\+Buffer\mbox{[}5\mbox{]}.

Tracks whether a read has been started, so that we know whether to start reads on new child calls. This only matters for the first read, because once a message comes in the child call becomes committed and there will be no new child calls.
