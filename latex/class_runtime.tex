Sets the processing state of an extension instance.

\begin{DoxyRemark}{Remarks}
Use this method to report the results of a lifecycle event handler.
\end{DoxyRemark}
If the lifecycle event failed \& the extension instance will no longer work correctly, use {\ttfamily \doxylink{class_runtime_aaca54f04dfefc0a200960459212e18f8}{Runtime.\+set\+Fatal\+Error}} instead.

To report the status of function calls other than lifecycle event handlers, use {\ttfamily console.\+log} or the Cloud \doxylink{class_functions}{Functions} logger SDK.
