\doxysubsubsubsubsection*{ACLs}

Cloud \doxylink{class_storage}{Storage} uses access control lists (ACLs) to manage object and bucket access. ACLs are the mechanism you use to share files with other users and allow other users to access your buckets and files.

To learn more about ACLs, read this overview on {\ttfamily \doxylink{}{Access Control}}.

See {\ttfamily \doxylink{}{Cloud Storage overview}} See {\ttfamily \doxylink{}{Access Control}}
