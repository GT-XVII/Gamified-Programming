\doxylink{class_stream_array_parser}{Stream\+Array\+Parser} processes array of valid JSON objects in random chunks through readable stream, and produces a stream of plain Javascript objects where it converted from the corresponding protobuf message instance.

The default JSON parser decodes the input stream under the following rules\+:
\begin{DoxyEnumerate}
\item The stream represents a valid JSON array (must start with a "{}\mbox{[}"{} and close with the corresponding "{}\mbox{]}"{}). Each element of this array is assumed to be either an array or an object, and will be decoded as a JS object and delivered.
\item All JSON elements in the buffer will be decoded and delivered in a stream.
\end{DoxyEnumerate}
