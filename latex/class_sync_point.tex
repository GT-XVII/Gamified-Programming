\doxysubsection{Detailed Description}
\doxylink{class_sync_point}{Sync\+Point} represents a single location in a \doxylink{class_sync_tree}{Sync\+Tree} with 1 or more event registrations, meaning we need to maintain 1 or more Views at this location to cache server data and raise appropriate events for server changes and user writes (set, transaction, update).

It\textquotesingle{}s responsible for\+:
\begin{DoxyItemize}
\item Maintaining the set of 1 or more views necessary at this location (a \doxylink{class_sync_point}{Sync\+Point} with 0 views should be removed).
\item Proxying user / server operations to the views as appropriate (i.\+e. apply\+Server\+Overwrite, apply\+User\+Overwrite, etc.) 
\end{DoxyItemize}
