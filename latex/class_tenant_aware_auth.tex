\doxysubsection{Detailed Description}
Tenant-\/aware {\ttfamily \doxylink{class_auth}{Auth}} interface used for managing users, configuring SAML/\+OIDC providers, generating email links for password reset, email verification, etc for specific tenants.

Multi-\/tenancy support requires Google Cloud\textquotesingle{}s Identity Platform (GCIP). To learn more about GCIP, including pricing and features, see the {\ttfamily \doxylink{}{\texorpdfstring{$\vert$}{|} GCIP documentation}}.

Each tenant contains its own identity providers, settings and sets of users. Using {\ttfamily \doxylink{class_tenant_aware_auth}{Tenant\+Aware\+Auth}}, users for a specific tenant and corresponding OIDC/\+SAML configurations can also be managed, ID tokens for users signed in to a specific tenant can be verified, and email action links can also be generated for users belonging to the tenant.

{\ttfamily \doxylink{class_tenant_aware_auth}{Tenant\+Aware\+Auth}} instances for a specific {\ttfamily tenant\+Id} can be instantiated by calling {\ttfamily \doxylink{class_tenant_manager_a181b30109f3224426456eaf4aa0b805a}{Tenant\+Manager.\+auth\+For\+Tenant}}.
