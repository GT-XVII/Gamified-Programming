\doxysubsection{Detailed Description}
Defines the tenant manager used to help manage tenant related operations. This includes\+: 
\begin{DoxyItemize}
\item The ability to create, update, list, get and delete tenants for the underlying project. 
\item Getting a {\ttfamily \doxylink{class_tenant_aware_auth}{Tenant\+Aware\+Auth}} instance for running \doxylink{class_auth}{Auth} related operations (user management, provider configuration management, token verification, email link generation, etc) in the context of a specified tenant. 
\end{DoxyItemize}
