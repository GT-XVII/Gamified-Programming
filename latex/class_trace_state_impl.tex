\doxysubsection{Detailed Description}
Trace\+State must be a class and not a simple object type because of the spec requirement (\href{https://www.w3.org/TR/trace-context/\#tracestate-field}{\texttt{ https\+://www.\+w3.\+org/\+TR/trace-\/context/\#tracestate-\/field}}).

Here is the list of allowed mutations\+:
\begin{DoxyItemize}
\item New key-\/value pair should be added into the beginning of the list
\item The value of any key can be updated. Modified keys MUST be moved to the beginning of the list. 
\end{DoxyItemize}
