\doxysubsection{Detailed Description}
\doxylink{class_a}{A} reference to a transaction.

The \doxylink{class_transaction}{Transaction} object passed to a transaction\textquotesingle{}s update\+Function provides the methods to read and write data within the transaction context. See \mbox{[}\doxylink{class_transaction_a4f222939dd40e6007415b81227cc3606}{run\+Transaction()}\mbox{]}{\ttfamily \doxylink{class_firestore_acf105d82306b948d6acb38b048f8cd15}{Firestore\#run\+Transaction}}.
