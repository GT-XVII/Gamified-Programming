\doxysubsection{Detailed Description}
\doxylink{class_a}{A} {\ttfamily \doxylink{class_unary_filter}{Unary\+Filter}} represents a restriction on one field value and can be used to refine the results of a {\ttfamily \doxylink{class_query}{Query}}. {\ttfamily \doxylink{class_unary_filter}{Unary\+Filter}}s are created by invoking {\ttfamily \doxylink{class_filter_af6be2934075d0920033f1d57a1608b11}{Filter\#where}} and can then be passed to {\ttfamily \doxylink{class_query_a09b549eea59e9d2d0d81b07b1b34bb77}{Query\#where}} to create a new {\ttfamily \doxylink{class_query}{Query}} instance that also contains this {\ttfamily \doxylink{class_unary_filter}{Unary\+Filter}}.
