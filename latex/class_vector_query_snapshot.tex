\doxysubsection{Detailed Description}
\doxylink{class_a}{A} {\ttfamily \doxylink{class_vector_query_snapshot}{Vector\+Query\+Snapshot}} contains zero or more {\ttfamily \doxylink{class_query_document_snapshot}{Query\+Document\+Snapshot}} objects representing the results of a query. The documents can be accessed as an array via the {\ttfamily docs} property or enumerated using the {\ttfamily for\+Each} method. The number of documents can be determined via the {\ttfamily empty} and {\ttfamily size} properties.
