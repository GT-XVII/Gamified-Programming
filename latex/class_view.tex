See the License for the specific language governing permissions and limitations under the License. \doxylink{class_a}{A} view represents a specific location and query that has 1 or more event registrations.

It does several things\+:
\begin{DoxyItemize}
\item Maintains the list of event registrations for this location/query.
\item Maintains a cache of the data visible for this location/query.
\item Applies new operations (via apply\+Operation), updates the cache, and based on the event registrations returns the set of events to be raised. 
\end{DoxyItemize}
