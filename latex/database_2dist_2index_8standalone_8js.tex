Helper to run some code but catch any exceptions and re-\/throw them later. Useful for preventing user callbacks from breaking internal code.

Re-\/throwing the exception from a set\+Timeout is a little evil, but it\textquotesingle{}s very convenient (we don\textquotesingle{}t have to try to figure out when is a safe point to re-\/throw it), and the behavior seems reasonable\+:


\begin{DoxyItemize}
\item If you aren\textquotesingle{}t pausing on exceptions, you get an error in the console with the correct stack trace.
\item If you\textquotesingle{}re pausing on all exceptions, the debugger will pause on your exception and then again when we rethrow it.
\item If you\textquotesingle{}re only pausing on uncaught exceptions, the debugger will only pause on us re-\/throwing it.
\end{DoxyItemize}
