Creates an RSA KEM API object for generating a secret asymmetric key.

The symmetric key may be generated via a call to \textquotesingle{}encrypt\textquotesingle{}, which will produce a ciphertext to be transmitted to the recipient and a key to be kept secret. The ciphertext is a parameter to be passed to \textquotesingle{}decrypt\textquotesingle{} which will produce the same secret key for the recipient to use to decrypt a message that was encrypted with the secret key.
