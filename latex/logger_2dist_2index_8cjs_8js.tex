The JS SDK supports 5 log levels and also allows a user the ability to silence the logs altogether.

The order is a follows\+: DEBUG \texorpdfstring{$<$}{<} VERBOSE \texorpdfstring{$<$}{<} INFO \texorpdfstring{$<$}{<} WARN \texorpdfstring{$<$}{<} ERROR
