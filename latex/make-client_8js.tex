Creates a constructor for a client with the given methods, as specified in the methods argument. The resulting class will have an instance method for each method in the service, which is a partial application of one of the \mbox{[}\doxylink{class_client}{Client}\mbox{]}{\ttfamily \doxylink{}{grpc.\+Client}} request methods, depending on {\ttfamily request\+Serialize} and {\ttfamily response\+Serialize}, with the {\ttfamily method}, {\ttfamily serialize}, and {\ttfamily deserialize} arguments predefined. 
\begin{DoxyParams}{Parameters}

\hline
{\em service\+Name} & The fully qualified name of the service \\
\hline
{\em class\+Options} & An options object. \\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
