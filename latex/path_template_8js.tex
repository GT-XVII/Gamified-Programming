function split\+Path\+Template (\begin{DoxyParamCaption}\item[{}]{data}{}\end{DoxyParamCaption})}

Split the path template by {\ttfamily /}. It can not be simply splitted by {\ttfamily /} because there might be {\ttfamily /} in the segments. For example\+: \textquotesingle{}a/b/\{a=hello/world\}\textquotesingle{} we do not want to break the brackets pair so above path will be splitted as \mbox{[}\textquotesingle{}a\textquotesingle{}, \textquotesingle{}b\textquotesingle{}, \textquotesingle{}\{a=hello/world\}\textquotesingle{}\mbox{]}
