function \+\_\+decode\+Pkcs7\+Data (\begin{DoxyParamCaption}\item[{}]{data}{}\end{DoxyParamCaption})}

Decodes PKCS\#7 Data. PKCS\#7 (RFC 2315) defines "{}\+Data"{} as an OCTET STRING, but it is sometimes an OCTET STRING that is composed/constructed of chunks, each its own OCTET STRING. This is BER-\/encoding vs. DER-\/encoding. This function transforms this corner-\/case into the usual simple, non-\/composed/constructed OCTET STRING.

This function may be moved to ASN.\+1 at some point to better deal with more BER-\/encoding issues, should they arise.

{\em data} & the ASN.\+1 Data object to transform. \\

Mac\+Data \+::= SEQUENCE \{ mac Digest\+Info, mac\+Salt OCTET STRING, iterations INTEGER DEFAULT 1 \} Note\+: The iterations default is for historical reasons and its use is deprecated. \doxylink{class_a}{A} higher value, like 1024, is recommended.

Digest\+Info is defined in PKCS\#7 as follows\+:

Digest\+Algorithm\+Identifier \+::= Algorithm\+Identifier

Algorithm\+Identifer \+::= SEQUENCE \{ algorithm OBJECT IDENTIFIER, parameters ANY DEFINED BY algorithm OPTIONAL \}

Digest \+::= OCTET STRING

Content\+Type \+::= OBJECT IDENTIFIER

Safe\+Contents \+::= SEQUENCE OF Safe\+Bag

Safe\+Bag \+::= SEQUENCE \{ bag\+Id BAG-\/\+TYPE.\&id (\{PKCS12\+Bag\+Set\}) bag\+Value \mbox{[}0\mbox{]} EXPLICIT BAG-\/\+TYPE.\&Type(\{PKCS12\+Bag\+Set\}\{@bag\+Id\}), bag\+Attributes SET OF PKCS12\+Attribute OPTIONAL \}

Wraps a private key and certificate in a PKCS\#12 PFX wrapper. If a password is provided then the private key will be encrypted.

An entire certificate chain may also be included. To do this, pass an array for the "{}cert"{} parameter where the first certificate is the one that is paired with the private key and each subsequent one verifies the previous one. The certificates may be in PEM format or have been already parsed by Forge.
