Reads the "{}common part"{} of an PKCS\#7 content block (in ASN.\+1 format)

This function reads the "{}common part"{} of the PKCS\#7 content blocks Encrypted\+Data and Enveloped\+Data, i.\+e. version number and symmetrically encrypted content block.

The result of the ASN.\+1 validate and capture process is returned to allow the caller to extract further data, e.\+g. the list of recipients in case of a Enveloped\+Data object.
