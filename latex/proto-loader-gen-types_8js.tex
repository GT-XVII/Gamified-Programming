This is a list of methods that are exist in the generic \doxylink{class_client}{Client} class in the g\+RPC libraries. Type\+Script has a problem with methods in subclasses with the same names as methods in the superclass, but with mismatched APIs. So, we avoid generating methods with these names in the service client interfaces. We always generate two service client methods per service method\+: one camel cased, and one with the original casing. So we will still generate one service client method for any conflicting name.
