\mbox{\hyperlink{pss_8js_a7f662015a42f9cce02297d822ce5f6c6}{pss}} create}

Creates a PSS signature scheme object.

There are several ways to provide a salt for encoding\+:


\begin{DoxyEnumerate}
\item Specify the salt\+Length only and the built-\/in PRNG will generate it.
\item Specify the salt\+Length and a custom PRNG with \textquotesingle{}get\+Bytes\+Sync\textquotesingle{} defined that will be used.
\item Specify the salt itself as a forge.\+util.\+Byte\+Buffer.
\end{DoxyEnumerate}


\begin{DoxyParams}{Parameters}
{\em options} & the options to use\+: md the message digest object to use, a forge md instance. mgf the mask generation function to use, a forge mgf instance. \mbox{[}salt\+Length\mbox{]} the length of the salt in octets. \mbox{[}prng\mbox{]} the pseudo-\/random number generator to use to produce a salt. \mbox{[}salt\mbox{]} the salt to use when encoding.\\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
a signature scheme object.
