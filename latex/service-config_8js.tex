function extract\+And\+Select\+Service\+Config (\begin{DoxyParamCaption}\item[{}]{txt\+Record}{, }\item[{}]{percentage}{}\end{DoxyParamCaption})}

Find the "{}grpc\+\_\+config"{} record among the TXT records, parse its value as JSON, validate its contents, and select a service config with selection fields that all match this client. Most of these steps can fail with an error; the caller must handle any errors thrown this way.

\hline

\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
The service configuration to use, given the percentage value, or null if the service config data has a valid format but none of the options match the current client.
