Default conversion options used for {\ttfamily \doxylink{}{Message\#to\+JSON}} implementations.

These options are close to proto3\textquotesingle{}s JSON mapping with the exception that internal types like Any are handled just like messages. More precisely\+:


\begin{DoxyItemize}
\item Longs become strings
\item Enums become string keys
\item Bytes become base64 encoded strings
\item (Sub-\/)Messages become plain objects
\item Maps become plain objects with all string keys
\item Repeated fields become arrays
\item NaN and Infinity for float and double fields become strings
\end{DoxyItemize}
