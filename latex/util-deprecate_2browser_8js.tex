function deprecate (\begin{DoxyParamCaption}\item[{}]{fn}{, }\item[{}]{msg}{}\end{DoxyParamCaption})}

Mark that a method should not be used. Returns a modified function which warns once by default.

If {\ttfamily local\+Storage.\+no\+Deprecation = true} is set, then it is a no-\/op.

If {\ttfamily local\+Storage.\+throw\+Deprecation = true} is set, then deprecated functions will throw an Error when invoked.

If {\ttfamily local\+Storage.\+trace\+Deprecation = true} is set, then deprecated functions will invoke {\ttfamily console.\+trace()} instead of {\ttfamily console.\+error()}.
